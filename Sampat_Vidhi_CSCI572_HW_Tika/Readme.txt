In the processfile implementation, the usinf PDFParser the corpus is parsed. For each parsed file in the corpus, a search is carried out for all the keywords and phrases mentioned in keywords.txt. Space is attached in front of and at the end of the keyword to reduce ambiguous results. Different statistics are calculated and the output and log files populated.

No special instructions to run the file.

An  Output file with information about No of files processed, No of files containing keyword(s), No of occurrences (count) of each keyword(s)
and a Log file with information about Filename of PDF file containing keyword, Timestamp (time when the file was processed) is created.
